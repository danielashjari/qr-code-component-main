### What I learned

This challenge is quite simple and it helped me to review baseics and reminded me how always use the easiest solotion. 
For example:

```html
<div class="card">
    <div class="QRcode">
      <img src="./images/image-qr-code.png" alt="QR code image">
    </div>
    ...
  </div>
```

```css
    .card {
        padding: 1rem;
        margin: 10vh 35vw;
        width: 18rem;
        height: 30rem;
        .QRcode {
            img {
                width: 100%;
            }
        }
    }
```
